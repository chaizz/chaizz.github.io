1 JSON.stringify(obj)

JSON编码的对象与JS对象的字面量有一些区别：

  • 字符串使用双引号，没有单引号或者反引号。
  • 对象的属性名称也是双引号的，是强制性的。

JSON支持以下的数据类型：

  • Objects {…}
  • Arrays […]
  • Strings ‘ ‘
  • numbers 1
  • boolean true false
  • null

JSON是语言无关的数据规范，所以一些属于JS的对象属性将会被忽略掉，例如：

  • 函数
  • Symbol
  • undefined

Tips：在使用JSON.stringify(obj) 时（仅包含单个参数），对象中不能有循环引用。
